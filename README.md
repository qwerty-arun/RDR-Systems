# RADAR Systems Programs in MATLAB

-  To detect the range of target using a FMCW radar system by simulating transmitter, receiver and beat signals in MATLAB.
-  To model a CW Radar to estimate the Doppler Shift and calculate target velocity and to implement pulse radar system to estimate the target range.
-  To develop a MATLAB program that computes the range, azimuth angle, and elevation angle of multiple targets, using implementations of mathematical functions.
-  To estimate the range of a target in the presence of clutter and noise using pulse radar and matched filtering in MATLAB.
-  To simulate and analyze the spectrogram of FMCW radar signals for multiple automotive radar systems in MATLAB.
-  Ambiguity Function Analysis for Single-Target Radar Detection.
-  Radar Target Detection and Range Estimation considering RCS.
-  Single-Target Radar Tracking Using Kalman Filter.
-  Simulation of MIMO Radar for Target Detection.
-  Adaptive Radar Target Detection Using CFAR.
